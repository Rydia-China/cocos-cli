# Platform Overview 
Google Play Instant is a lightweight app and game trial experience for Android developers.
It allows users to try an app or game without completing a full installation, lowering the barrier to entry and improving conversion efficiency. For game developers, this is especially useful for playable previews, onboarding experiences, or lightweight game entry points that drive installs.Combined with Google Play distribution, storefront exposure, and the Android toolchain, it helps create a smooth user journey from instant trial to install and long-term retention.
